# Code and Data

## Files
- Model_1.ipynb — runs Model 1 simulations and produces figures  
- Model_2.ipynb — runs Model 2 simulations and produces figures  
- Graph_parameters.ipynb — prepares parameters and data

- N40_y2.4.edgelist — example network file  

## Run in Google Colab
1. Open https://colab.research.google.com  
2. Upload a notebook (e.g., Model_1.ipynb)  
3. Upload the file: N40_y2.4.edgelist  
4. Run all cells (Runtime → Run all)
